# Data-205-Capstone Project-Ike Charistan

# Overview
This capstone project investigates whether public financial investments distributed through Maryland state grants and loans are associated with measurable economic outcomes at the county level. Specifically, the analysis focuses on Montgomery County and Prince George’s County—two of Maryland’s largest neighboring jurisdictions with distinct economic structures and labor market outcomes.
